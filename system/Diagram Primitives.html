A reusable diagram vocabulary

Six primitives.

The recurring diagram types a teaching deck reaches for, built once as clean, label‑swappable objects. Drop one into a slide and relabel it — never re‑specify its geometry. Each is composed from the same six‑part grammar, so it lands on the substrate and reads at a glance on a projector and on a phone.

GREEN primary · default
BLUE secondary
PURPLE tertiary

Every primitive carries its accent on one token, so it is swappable in one move — set --c (or swap the t-GRNd / dgm-GRN class) to blue or purple. Shown here in the default green. Note: the brief asked for blue/orange/green; this system’s palette is locked and orange is retired, so the third swap is purple per the green→blue→purple hierarchy.

Use when — you need to find an open gap in a crowded field. Plot how it gets solved today, highlight the empty quadrant, drop your bet in it.

02 · Nested frames accent: centre + layer in focus
Use when — you want to show the layers of context around one person. Centre is who you serve; each ring is a layer further out.

THE USER

ACCENT → THE CENTRE FIGURE + THE LAYER IN FOCUS
03 · Stage flow accent: the output stage
Use when — you are walking through an activity sequence. Four stages, three arrows; each node carries a label and a small time / output sub‑line.

04 · Fill‑in‑the‑blank formula accent: blanks + key word
Use when — you want to compress an insight to one line. Fixed connective text in neutral, three swappable blanks in the accent, one marked key word.
PRIMITIVE 04 · FILL‑IN FORMULA
who can’t do the job because the obstacle, so today nothing really fits.
the obstacle ← the riskiest word. Mark it, test it first.
ACCENT → THE THREE BLANKS + THE KEY‑WORD MARK
05 · Stat / recap cards accent: the headline stat
Use when — you are recapping with numbers. A row of three: each a big numeral over a short label. Tone ranks them; accent the one that matters most.

06 · Before / after delta accent: the after frame
Use when — you are showing a change. Left is what we had; right is what changed — and the right column is sized larger so it visibly carries the point.
PRIMITIVE 06 · BEFORE / AFTER DELTA
What changed, and why it matters.
WHAT WE HAD
A clever build, shipped on a hunch. We guessed at the need and hoped the market agreed.
WHAT CHANGED
A need we heard in real words, from real people. The same effort, aimed at a problem we could name — and a sentence anyone on the team can repeat back.
ACCENT → THE AFTER FRAME (THE LARGER, GREEN‑EDGED COLUMN)
Six objects, one grammar. Relabel any of them in place — swap the text, swap one accent token (green → blue → purple) — and the geometry, spacing and type stay on the substrate. Press L to lint, G for the grid.
